# Rivalry Atlas

Rivalry Atlas is a beginner-friendly soccer website built with HTML, CSS, and JavaScript as part of the CodePath WEB101 course. The project explores some of the most iconic football rivalries while helping develop core front-end web development skills such as responsive layout design, styling, and JavaScript interactivity.

The website includes multiple informational sections, interactive features, images, and a clean user experience focused on accessibility and readability.

---

## Live Demo

[View the Live Website](https://christian101gtz.github.io/CGomez_WEB101/)

---

## Features

- Responsive multi-section website layout
- Navigation bar for easy page navigation
- Dark mode toggle built with JavaScript
- Rivalry Showcase section with images and descriptions
- Interactive RSVP form
- Organized HTML structure and reusable styling
- Beginner-friendly and clean visual design
- Sections highlighting famous football rivalries such as:
  - El Clásico
  - USA vs Mexico
  - Additional rivalry showcases
